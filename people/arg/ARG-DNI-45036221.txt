# Proyectos Destacados
## The MovieVerse - DESARROLLO WEB FULLSTACK
- Descripción: TheMovieVerse es una aplicación web desarrollada con React.js y Express.js que permite a los usuarios explorar información sobre películas y series. Utiliza una API propia para obtener datos actualizados sobre estrenos, tendencias y detalles de cada título.
- Tecnologías: Express.js - React.js
- Link: https://github.com/JuantMartinez17/TheMovieVerse

## The Knights Odissey - DESARROLLO Y DISEÑO DE VIDEOJUEGOS
- Descripción: es un juego de plataformas 2D desarrollado en Godot 4.3. El juego busca demostrar la implementación de mecánicas clave como animaciones fluidas, persistencia de datos, sonidos envolventes, diseño de niveles y el uso de patrones de diseño para organizar el código eficientemente.
- Tecnologías: Godot - GDScript
- Link: https://github.com/JuantMartinez17/TheKnightsOdissey_OnGodot

## MealMind - DESARROLLO DE APLICACIONES MOVILES ANDROID
- Descripción: MealMind es una aplicación Android desarrollada en Kotlin que permite a los usuarios buscar recetas basadas en ingredientes y guardarlas como favoritas. Utiliza la API de Spoonacular para obtener información detallada sobre las recetas, incluyendo datos nutricionales.
- Tecnologías: Kotlin - XML
- Link: https://github.com/JuantMartinez17/MealMind
